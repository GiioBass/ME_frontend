# Frontend Technology Stack - Mystic Explorers

The frontend is a modern, reactive single-page application (SPA) designed with a **Cyberpunk/Sci-Fi aesthetic**.

## Core Technologies
- **Language**: [TypeScript](https://www.typescriptlang.org/) - Strong typing for better developer experience and reliability.
- **Library**: [React 19](https://react.dev/) - Modern UI library with Hooks and Concurrent Features.
- **Build Tool**: [Vite 7](https://vitejs.dev/) - Next-generation frontend tooling.

## Styling & UI
- **CSS Framework**: [Tailwind CSS 4](https://tailwindcss.com/) - Utility-first styling with high performance.
- **Icons**: [Lucide React](https://lucide.dev/) - Beautifully simple pixel-perfect icons.
- **Animations**: CSS Transitions & Tailwind variants for micro-interactions.

## Application Architecture
- **State Management**: Custom React Hooks (`useGameEngine`) for game loop and session management.
- **API Communication**: [Axios](https://axios-http.com/) - Promise-based HTTP client for the browser.
- **Dynamic HUD**: Component-based architecture (`MainGameHUD`, `BioMetrics`, `RadarPanel`).

## Development
- **Linting**: [ESLint 9](https://eslint.org/).
- **Type Checking**: Vite-integrated TypeScript compiler.
- **Environment**: Node.js with NPM.
